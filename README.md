#PROJECT TITLE :hospitality-revenue-analysis-powerbi
Hospitality analytics project using Power BI to analyze revenue, occupancy, ADR and RevPAR metrics.

#PROJECT OVERVIEW
This project analyzes hospitality business performance using Power BI. The dashboard provides insights into revenue, occupancy, ADR, RevPAR, booking trends, and overall hotel performance.
#OBJECTIVE
- Analyze hotel performance
- Monitor revenue trends
- Compare occupancy rates
- Support business decision-making
  ## Tools Used
- Power BI
- DAX
- Excel
- Data Modeling
 ## KPIs
- Revenue
- Occupancy %
- ADR (Average Daily Rate)
- RevPAR
- Revenue Growth
 ## Dashboard Features
- Revenue Analysis
- Booking Trends
- Occupancy Analysis
- Interactive Filters
- KPI Cards
- Monthly Performance
